# Trinh Hoang Tu (thtcsec) - Portfolio Website

A modern, highly interactive, and responsive portfolio website showcasing my expertise in **Cloud Computing, AI/Computer Vision, and Full-Stack Development**. Built with React, TypeScript, TailwindCSS, and deployed on GitHub Pages.

## 🚀 Live Demo
**[https://thtcsec.github.io](https://thtcsec.github.io)**

## ✨ Features

- ⚡ **Lightning Fast** - Built with Vite for rapid development and HMR.
- 🎨 **Modern Aesthetics** - Clean, glass-morphism designs powered by **TailwindCSS** and **Framer Motion** animations.
- 🌓 **Theming** - Persistent Dark/Light mode toggle.
- 📊 **Dynamic Diagrams** - Native integration of **Mermaid.js** for rendering complex Cloud architecture sequence diagrams directly on the UI.
- 📱 **Fully Responsive** - Flawless experience across all mobile, tablet, and desktop devices.
- 🔍 **SEO & Tracking** - Pre-configured Open Graph tags, Schema.org metadata, and Google Analytics 4 (GA4).
- 📲 **PWA Ready** - Fully installable as an app with custom manifest and service workers.
